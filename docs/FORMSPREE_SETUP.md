# Formspree setup guide (for the page owner)

This guide is for the **site owner**. It explains only what you need to do **on Formspree** so the enquiry form works and you receive every submission by email.

---

## 1. Create your Formspree account

1. Go to **https://formspree.io** in your browser.
2. Click **“Get started”** or **“Sign up”**.
3. Sign up using the **email where you want to receive enquiries** (e.g. **thibault@epevents.net**).
4. If Formspree sends a verification email, open it and click the link to confirm your address.

---

## 2. Create a new form

1. Log in at **https://formspree.io**.
2. Click **“+ New form”** (or **“New form”**).
3. Give it a name (e.g. **“Khaima enquiries”**).
4. Click **“Create form”**.
5. You’ll see a URL like: **`https://formspree.io/f/xxxxxxxx`**
   - **Copy that full URL** (or at least the part after `/f/`, e.g. `xxxxxxxx`).
   - **Send it to your developer** so they can connect it to the website. You don’t need to change any code yourself.

---

## 3. Turn on email notifications (receive full answers)

1. In Formspree, open **your form** (click its name).
2. Go to **“Settings”** (gear icon or “Settings” in the menu).
3. Find **“Notifications”** or **“Email notifications”**.
4. Make sure **“Email notifications”** is **on** (enabled).
5. Set the notification email to the address where you want to receive each submission (e.g. **thibault@epevents.net**). If you signed up with that email, it’s usually already set.
6. Save any changes.

After this, **every time someone submits the enquiry form**, you’ll get an email to that address with:
- All the answers (name, email, event type, location, guests, dates, meals, dietary, notes),
- Date and time of the submission.

---

## 4. View submissions in Formspree (optional)

1. Log in at **https://formspree.io**.
2. Click your form’s name.
3. Open the **“Submissions”** (or “Inbox”) tab to see all enquiries in one place. You can export them if needed.

---

## Summary

| Step | What you do |
|------|------------------|
| 1 | Sign up at formspree.io with the email that should receive enquiries. |
| 2 | Create a new form, copy its URL (e.g. `https://formspree.io/f/xxxxxxxx`), and send it to your developer. |
| 3 | In Form form → Settings → Notifications: turn on email notifications and set the correct email. |
| 4 | (Optional) Use the Submissions tab to view and export all enquiries. |

You don’t need to edit the website or any code. The developer will use the form URL you send them to connect the form on the site.
